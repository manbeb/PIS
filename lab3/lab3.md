# Лабораторная работа №3 – Бизнес-процессы предметной области (AS-IS)

Выявить и формализовать ключевые бизнес-процессы предметной области в текущем состоянии (AS-IS), определить их структуру, участников и проблемы.

Работа выполняется по ранее выбранной предметной области.

## Задание №1

Определить:
- основной (ключевой) процесс системы;
- 2 вспомогательных процесса.

Для каждого процесса указать:
- цель процесса;
- инициатор;
- входные данные;
- выходные данные;
- участников;
- ожидаемый результат.

## Задание №2

Построить BPMN-диаграмму основного процесса и BPMN-диаграмму одного из вложенных процессов в основной.

## Задание №3

Проанализировать основной процесс и выявить возможные проблемы, такие как:
- узкие места;
- дублирование операцией;
- ручные операции;
- задержки;
- риски;
- избыточность ввода данных.

Оформить в виде таблицы:

| Возможная проблема | Причина | Последствие |
|--------------------|---------|-------------|
|                    |         |             |

## Результаты

Сдать преподавателю и загрузить в репозиторий:
- текстовое описание процессов (любой формат и оформление);
- две диаграммы BPMN (PDF);
- таблицу проблем (любой формат и оформление).
